# CLR-UDT-App
User Defined Types .NET console application

Types to be defined

* Book
  - author name
  - title
  - year of publication

* Student
  - name
  - field of study
  - year 

* Movie
  - title
  - type
  - release date

* Car
  - brand
  - model
  - horsepower

* Fruit
  - name
  - color
  - calories

* Bank Account
  - user ID
  - balance
